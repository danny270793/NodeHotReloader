# HotReloader

Re-execute command when a file changes

## Instalation

Install package from public registry

```bash
npm install @danny270793/hotreloader
```

## Example

Import the library

```json
{
    ...
    "scripts": {
        "start:watch": "@danny270793/hotreloader node ./src/index.js"
    }
    ...
}
```
